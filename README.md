## 📋 Sobre o projeto

Este projeto simula a página de vendas de uma agência de marketing digital voltada para pequenos negócios locais. A proposta era criar uma estrutura que qualquer negócio pudesse adaptar rapidamente — trocando textos e imagens — sem precisar reescrever a base de código.

O foco do projeto foi:
- Uma hierarquia visual clara, guiando o visitante do problema até a ação (falar no WhatsApp)
- Um design que não parecesse um template genérico de IA, com paleta e tipografia autorais
- Zero dependências pesadas — sem frameworks de JS, sem build step

## ✨ Funcionalidades

- **Hero de impacto** com proposta de valor e prova social imediata
- **Seção de serviços** com cards e ícones
- **Prova social** com números e depoimentos de clientes
- **Galeria** de trabalhos/resultados
- **CTA final** direcionado para conversão
- **Botão flutuante de WhatsApp** com animação de pulso, sempre visível
- **Scroll reveal** — elementos aparecem suavemente conforme o usuário rola a página
- **Totalmente responsivo** — mobile, tablet e desktop
- Respeita `prefers-reduced-motion` para acessibilidade

## 🛠️ Tecnologias

- **HTML5** — estrutura semântica
- **Tailwind CSS** (via CDN) — estilização utilitária, com tema customizado (cores e fontes próprias configuradas via `tailwind.config`)
- **JavaScript puro** — apenas o necessário (Intersection Observer para o efeito de scroll reveal)
- **Google Fonts** — Space Grotesk (display) + Inter (texto)

Nenhuma dependência de build, bundler ou framework — o arquivo roda sozinho em qualquer navegador.
